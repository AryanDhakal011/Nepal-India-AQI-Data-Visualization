numpy==1.26.4
matplotlib==3.9.0
seaborn==0.13.2
plotly==5.22.0
folium==0.16.0
streamlit==1.35.0
streamlit-folium==0.20.0

### How to run the code ###
- Download the Folder 
- Open the folder in an IDE like VsCode 
- Open the Terminal 
- Go inside the AQI Code Folder (cd FOLDER_NAME') 
- Check Python Version (python --version) Note: Requires above 3.10
- Install Packages: pip install pandas numpy matplotlib seaborn plotly folium streamlit streamlit-folium
- Run Streamlit: streamlit run dashboard.py 

You will now see the AQI Visualization Dashboard on our default browser 
All visualization pictures are in the 'Output' folder.
